# Course
# 简介
1. 这是一个基于J2EE的MVC模式开发实践的校园网络课程管理系统
2. 运用bootstrap框架开发移动响应式
3. 运行环境：
* IDE:MyEclipse
* 数据库:MySql
* Tomcat:Tomcat 7.0
# 部署环境
1. 在开发环境新建一个Web Project,取名为Course
2. 复制项目目录下的src和WebRoot文件夹内容，粘帖进项目
3. 打开WebRoot->WEB-INF->lib，将里面的所有包右击加进java运行环境中
4. 将项目中的sql 文件导进MySql 中
5. 把项目配置服务器中即可运行项目
